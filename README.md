# 📝 Conversational Text Summarization using T5

## 📌 Overview
This project implements an abstractive text summarization system using the T5 (Text-to-Text Transfer Transformer) model. 

The model is fine-tuned on **general conversational data**, enabling it to generate concise summaries from dialogues, chats, and informal text. This makes it useful for applications like chat summarization, meeting notes, and customer support logs.

---

## 🚀 Features
- Fine-tuned T5 model on conversational datasets
- Handles informal and dialogue-style text
- Generates coherent and context-aware summaries
- Supports long input sequences
- Built using Hugging Face Transformers

---

## 🧠 Model Details
- Model: T5 (Text-to-Text Transfer Transformer)
- Base model: `t5-small`
- Fine-tuning domain: **General conversations / dialogues**
- Task: Abstractive text summarization
